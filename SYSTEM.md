## The Color Engine (Most Critical Chain)

Everything flows from **matugen**. One wallpaper change recolors the entire system.

```
Wallpaper change (awww)
    └── matugen image "$selected_image"
            ├── templates/hyprland.lua   → ~/.config/hypr/hyprland/colors.lua
            │       └── post_hook: hyprctl reload
            │               └── sets active_border + inactive_border in general.lua
            │
            ├── templates/waybar.css     → ~/.config/waybar/colors.css
            │       └── no post_hook (waybar has reload_style_on_change: true)
            │               └── @import "./colors.css" at top of styles.css
            │
            ├── templates/kitty.conf     → ~/.config/kitty/colors.conf
            │       └── post_hook: pkill -SIGUSR1 kitty
            │               └── include colors.conf at top of kitty.conf
            │
            ├── templates/rofi.rasi      → ~/.config/rofi/colors.rasi
            │       └── all rofi files: @import "./colors.rasi"
            │
            ├── templates/swaync.css     → ~/.config/swaync/theme.css
            │       └── post_hook: swaync-client --reload-css
            │               └── @import "theme.css" at top of style.css
            │
            ├── templates/wlogout.css    → ~/.config/wlogout/colors.css
            │       └── @import "$HOME/.config/wlogout/colors.css" in style.css
            │
            ├── templates/cava.ini       → ~/.config/cava/themes/matugen
            │       └── post_hook: pkill -USR1 cava
            │
            ├── templates/gtk.css        → ~/.config/gtk-3.0/colors.css
            │                            → ~/.config/gtk-4.0/colors.css
            │       └── post_hook: gsettings sets adw-gtk3-dark + prefer-dark
            │
            ├── templates/vicinae.toml   → ~/.local/share/vicinae/themes/matugen.toml
            │       └── post_hook: vicinae theme set matugen
            │
            ├── templates/qt.conf        → ~/.local/share/color-schemes/Matugen.colors
            │       └── picked up by qt5ct / qt6ct
            │
            └── templates/matugen.css    → ~/.config/matugen/matugen.css
                    └── reference/preview only, no consumers
```

## Wallpaper Change — Three Paths

### Path 1 — Random wallpaper (SUPER + ALT + P)
```
keybindings.lua → change-wall.sh ~/Pictures/Wallpapers
    ├── finds random image (avoids current wallpaper)
    ├── awww img -t grow (animated grow transition)
    ├── matugen image "$selected_image" (full system recolor)
    └── cp "$selected_image" ~/.cache/lock_background
```

### Path 2 — Pick wallpaper (SUPER + SHIFT + P)
```
keybindings.lua → vicinae awww-switcher/wpgrid
    └── vicinae settings.json preferences:
            ├── wallpaperPath: Pictures/Wallpapers
            ├── colorGenTool: matugen
            ├── postCommand: cp ${wallpaper} ~/.cache/lock_background
            └── transitionType: outer
```

### Path 3 — On login (autostart.lua)
```
autostart.lua
    ├── uwsm app -- awww-daemon
    └── sleep 0.3; restore-wall.sh
            ├── awww restore (restores last wallpaper state)
            └── matugen image "$image_path" (regenerates all colors)
```

## Hyprland Config Structure

```
~/.config/hypr/
├── hyprland.lua          ← entry point, requires all modules
└── hyprland/
    ├── general.lua       ← env vars, look/feel, blur, animations, input, gestures
    │                        requires("hyprland.colors") at top
    ├── colors.lua        ← GENERATED by matugen (active_border, inactive_border)
    ├── autostart.lua     ← all apps launched on Hyprland start via uwsm
    ├── keybindings.lua   ← all keybinds, defines terminal/browser/editor/fileManager
    ├── rules.lua         ← window rules, workspace rules, XWayland fixes
    │                        NOTE: toggle-noscreenshare.sh directly edits this file
    ├── monitors.lua      ← display config (eDP-1, highres, scale 1)
    └── hypridle.conf     ← idle timeout chain (separate, not Lua)
    └── hyprlock.conf     ← lock screen layout (separate, not Lua)
```

## Hypridle Timeout Chain

```
10 min (600s)  → brightnessctl dim to 20% (saves current brightness)
13 min (780s)  → notify-send "You are idle!" warning
15 min (900s)  → brightnessctl restore + loginctl lock-session (hyprlock)
16 min (960s)  → hyprctl dpms off (display off)
18 min (1080s) → systemctl suspend
```

**⚠️ Warning:** Poweroff and suspend button positions are hardcoded pixel values.
On a different monitor width they will be misaligned.
Fix: replace `position = 840, 100` with dynamic `(monitor_w * 0.58)` expressions.

## wlogout Launch Chain

```
SUPER + X → logout-launch.sh
    ├── detects if wlogout running → kills it (toggle behavior)
    ├── reads monitor height: hyprctl -j monitors | jq
    ├── calculates: mgn, hvr (margins based on monitor height + scale)
    ├── calculates: fntSize (2% of monitor height)
    ├── reads hypr rounding: hyprctl -j getoption decoration:rounding
    ├── calculates: active_rad (rounding * 5), button_rad (rounding * 8)
    ├── envsubst fills ${mgn} ${hvr} ${active_rad} ${button_rad} into style.css template
    └── wlogout -b 6 --layout layout --css <(evaluated CSS)
```

Colors come from `~/.config/wlogout/colors.css` (generated by matugen).
Icons: `~/.config/wlogout/icons/` (lock, logout, suspend, poweroff, hibernate, reboot).

## toggle-noscreenshare.sh Mechanism

```
SUPER + H → toggle-noscreenshare.sh
    ├── target file: ~/.config/hypr/hyprland/rules.lua
    ├── toggles these two lines (comment ↔ uncomment):
    │   hl.layer_rule({ match = { namespace = "^(.*)$" }, no_screen_share = true })
    │   hl.window_rule({ match = { initial_class = "^(.*)$" }, no_screen_share = true })
    ├── sends notify-send with Window/Layer status
    └── hyprctl reload
```

**⚠️ Warning:** Directly edits `rules.lua`. If you rename or restructure rules.lua, this script breaks.
Always keep those two lines present (commented out by default) in rules.lua.

## Screenshot Pipelines

### Local annotated (SUPER + SHIFT + S)
```
grim -g "$(slurp -d)" -t ppm - | satty --filename - --actions-on-enter save-to-clipboard,save-to-file,exit
```

### Zipline upload (SUPER + PRINT)
```
grim → satty → /tmp/file.png → curl POST $ZIPLINE_URL/api/upload
    Headers: Authorization, x-zipline-format: random, x-zipline-folder: $ZIPLINE_FOLDER_ID
    Response: .files[0].url → wl-copy to clipboard
    Requires: ~/.config/bin/.env with ZIPLINE_URL, ZIPLINE_API_KEY, ZIPLINE_FOLDER_ID
```

### Chibisafe upload (screenshot-chibisafe.sh)
```
grim → satty → /tmp/file.png → curl POST $CHIBISAFE_URL/api/upload
    Headers: x-api-key: $CHIBISAFE_API_KEY
    Response: .url → wl-copy to clipboard
    Requires: ~/.config/bin/.env with CHIBISAFE_URL, CHIBISAFE_API_KEY
```

## Screen Recording (record.sh)

```
SUPER + SHIFT + R → record.sh
    ├── if recording active → stop (SIGINT to gpu-screen-recorder)
    ├── if not recording:
    │   ├── optional: --with-audio (merges output + input)
    │   ├── optional: --with-webcam (ffplay overlay from /dev/video0)
    │   ├── scope: output (select monitor via slurp) or region (select area)
    │   └── gpu-screen-recorder -w [scope] -f 60 -c mp4 -o ~/Videos/Recordings/
    └── pkill -RTMIN+8 waybar (updates recording indicator)
```

## Fresh Install Checklist

```
1.  Install Arch + Hyprland base
2.  Clone dotfiles repo (bare git method)
3.  yay -S --needed - < ~/.config/pkglist.txt
4.  Install fonts manually if missing (JetBrainsMono Nerd, Inter, Noto Sans)
5.  Set up ~/.config/bin/.env (API keys for screenshot upload)
6.  Run: matugen image ~/Pictures/Wallpapers/[any wallpaper]
7.  Run: nwg-look -a (apply GTK theme)
8.  Reboot or start Hyprland via uwsm
9.  Check: hyprctl monitors (verify eDP-1 exists, else update monitors.lua)
10. Check: hyprlock poweroff/suspend button positions (update if not 1080p)
```
